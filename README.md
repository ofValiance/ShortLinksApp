# Описание

Консольное приложение для создания и управления короткими ссылками с возможностью установки лимита переходов и автоматическим удалением по истечении срока действия.

+ Создание уникальных коротких ссылок
+ Возможность установки максимального количества переходов по ссылке
+ Идентификация пользователей с UUID без необходимости авторизации и права доступа
+ Уведомления пользователей через сообщения о состоянии ссылок
+ Автоматическое удаление ссылок через 24 часа после создания
+ Фоновая очистка истекших ссылок каждые 30 секунд

# Классы

## ShortLink.java
Модель короткой ссылки
+ Хранение данных и состояния ссылки

## User.java
Модель пользователя
+ Идентификация по UUID
+ Хранение списка созданных ссылок

## ShortLinkManager.java
Логика приложения, управления пользователями и ссылками
+ Генерация уникальных ссылок
+ Управление пользователями и ссылками
+ Проверка состояния ссылок
+ Фоновая очистка истекших ссылок
+ Взаимодействие с браузером

## ShortLinksApp.java
Консольный интерфейс
+ Интерактивное меню
+ Обработка пользовательского ввода
+ Управление сессиями пользователей
+ Запуск фоновых задач

# Установка и запуск

## Jar
Запустить ShortLinksApp.jar в директории \jar командой:
java -jar "ShortLinksApp.jar"

## Ручная компиляция

В директории \javac запустить compile.bat. После успешной компиляции запустить приложение через run.bat.
